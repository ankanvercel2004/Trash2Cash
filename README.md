# ♻️ Trash2Cash

**Trash2Cash** is a sustainable marketplace that connects homeowners looking to discard waste with collectors who can reuse or recycle it — turning trash into opportunity.

Built with ❤️ using **Next.js**, **Firebase**, and **Tailwind CSS**.

---

## 🚀 Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) in your browser.

---

## ✨ Features

- 🔐 Firebase Authentication (Email/Password + Google)
- 🗑️ Full CRUD for waste listings
- 📊 Dashboard for listings & transactions
- 💅 Smooth UI with Tailwind & Animations

---

## 🧠 Tech Stack

- **Framework**: Next.js
- **Styling**: Tailwind CSS
- **Backend**: Firebase Firestore & Auth
- **Hosting**: Vercel

---

## 🛠 Deployment

Easily deploy using [Vercel](https://vercel.com):

```bash
npm run build
npm start
```
